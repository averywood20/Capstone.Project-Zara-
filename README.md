# Capstone.Project-Zara-
Final project for ITIS 4390 - Capstone design class. 

Overview
An improved Zara website designed with a strong focus on simplicity and easy navigation. The layout is uncluttered and user-friendly, allowing users to quickly find what they need without distractions.

Running this Project locally

1. Clone the Repository
   https://github.com/averywood20/Capstone.Project-Zara-

2. Open the Project Folder
   cd https://github.com/averywood20/
   
4. Run website
   Use live server in VScode
   Click on index.html
   Click on "Go Live" (Have Live Server Installed)
